P-Score Explained

The Pythia **P-Score** is a holistic measure of a website's performance and quality, ranging from 0 to 100. It is a weighted average of ten underlying **Component Metrics**, designed to give a comprehensive verdict on a site's digital health. A high P-Score indicates a fast, accessible, secure, and modern website.

P-Score Breakdown

  • 90-100: **Oracle Tier** - Exceptional performance.
  • 70-89: **Prophet Tier** - Great performance, minor room for improvement.
  • 50-69: **Seeker Tier** - Moderate performance, significant improvements needed.
  • 0-49: **Labyrinth Tier** - Poor performance, urgent attention required.

Component Metrics

The P-Score is calculated based on 10 distinct metrics, each scored out of 100. Improving these components directly raises your overall P-Score.

Karpov Speed

Measures load time, resource count, page weight, and optimization techniques.

Vortex Access

Evaluates image alt text, ARIA labels, semantic HTML, and accessibility features.

Nova Scale

Checks for CDN usage, caching headers, and compression.

Aether Tech

Detects WebAssembly, Service Workers, ES6 modules, and modern frameworks.

Pulse SEO

Analyzes meta tags, Open Graph, Twitter Cards, and SEO basics.

Eden Efficiency

Scores based on total page weight. Smaller sites score higher.

Helix Privacy

Counts trackers and checks security headers (HSTS, CSP, X-Frame).

Echo Green

Checks if hosting uses renewable energy via Green Web Foundation API.

Quantum Quality

Checks for code quality indicators and modern web practices.

Nexus Mobile

Evaluates viewport meta tags and media queries.

About Pythia

Pythia is a comprehensive website performance oracle designed to give you a single, holistic **P-Score** for any site on the web. It goes beyond traditional speed tests by evaluating ten critical dimensions of web health, from load speed and accessibility to security and green hosting.

The Mission

Our mission is to simplify web performance analysis. By consolidating complex metrics into one easy-to-understand score, we empower developers, marketers, and site owners to quickly diagnose and improve their digital presence.

Technology Stack

Pythia is built using modern web technologies, including a Cloudflare Workers backend for fast, serverless scanning, and a Tailwind CSS front-end for responsive design.
