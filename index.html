What is the P-Score?

The P-Score (Performance Score) is Pythia's unified metric that distills complex website analysis into a single, actionable number from 0-100. Think of it as a health checkup for your digital presence, measuring website performance across multiple critical dimensions.

How It Works

Pythia analyzes 10 distinct aspects of your website‚Äîfrom lightning-fast loading times to privacy practices‚Äîand intelligently weighs them based on what matters most to modern users and search engines. Our algorithm evaluates site speed, accessibility, SEO optimization, security headers, and sustainability practices to generate one comprehensive score.

Weighting Philosophy

The P-Score intelligently balances multiple factors for optimal website performance measurement:

  • Performance (20%) - Speed and responsiveness
  • Accessibility (20%) - Inclusive design for all users
  • SEO (15%) - Search engine optimization
  • Privacy & Security (15%) - User protection
  • Mobile Experience (10%) - Responsive design
  • Sustainability (7%) - Environmental impact
  • Infrastructure (7%) - CDN and scalability
  • Code Quality (3%) - Best practices
  • Modern Technology (2%) - Future-readiness
  • Efficiency (1%) - Page weight optimization

üîß P-Score Components

Karpov Speed (20%)

Measures page load time, resource optimization, and rendering performance. Evaluates Time to Interactive, First Contentful Paint, and total page weight.

Vortex Accessibility (20%)

Evaluates inclusive design: alt text on images, ARIA labels for screen readers, semantic HTML structure, and keyboard navigation support.

Pulse SEO (15%)

Analyzes meta tags, Open Graph cards for social sharing, Twitter Cards, canonical URLs, and search engine optimization basics.

Helix Privacy (15%)

Tracks third-party scripts, analytics trackers, and security headers (HSTS, CSP, X-Frame-Options).

Nexus Mobile (10%)

Assesses mobile responsiveness, viewport configuration, media queries, and touch optimization.

Echo Sustainability (7%)

Checks green hosting and carbon-conscious practices via Green Web Foundation API.

Nova Infrastructure (7%)

Checks CDN usage, caching strategies, HTTP/2 support, and compression.

Quantum Code Quality (3%)

Evaluates code quality, modern practices, SRI integrity checks, and web standards adherence.

Aether Technology (2%)

Detects modern frameworks, WebAssembly, Service Workers, ES6 modules, and next-gen image formats.

Eden Efficiency (1%)

Measures total page weight and resource usage. Leaner sites load faster.
